## 特殊格式说明

### 交互式代码块
通过在普通代码块的语言后面加上 `runner` 启用交互式代码块，仅支持 JavaScript 和 Python。
````md
```javascript runner
// ...
```
````

````md
```python runner
// ...
```
````

### 字数和阅读时间
通过 Frontmatter： `readingTime: boolean` 来控制是否显示字数和阅读时间，默认 true。
```
---
readingTime: false
---
```
